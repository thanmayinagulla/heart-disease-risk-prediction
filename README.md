# ❤️ Heart Disease AI Prediction System

This project uses Machine Learning to predict heart disease risk.

## Features
- Streamlit UI
- ML Model (Random Forest / XGBoost)
- Probability-based prediction
- PDF report generation

## Tech Stack
Python, Streamlit, Scikit-learn, Joblib, Reportlab

## Run Project
streamlit run app.py
