# ❤️ Heart Disease Risk Prediction AI

A Machine Learning web application that predicts the risk of heart disease using patient health parameters. Built with **Python**, **Scikit-learn**, and **Streamlit**.

## 🚀 Features

- Heart Disease Risk Prediction
- Risk Probability Analysis
- Personalized Diet Recommendations
- Lifestyle & Health Care Suggestions
- Downloadable PDF Health Report

## 🛠️ Tech Stack

- Python
- Streamlit
- Scikit-learn
- NumPy
- Joblib
- ReportLab

## ▶️ Run the Project

```bash
pip install -r requirements.txt
streamlit run app.py
```
